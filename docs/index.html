Getting Started
01

Install Ollama

Download the Windows installer from ollama.com/download. Installs silently and starts a background service on port 11434.

cmd
# Verify (open a fresh terminal)
ollama --version

# If not recognized ‚Äî add to PATH
set PATH=%PATH%;C:\Users\<you>\AppData\Local\Programs\Ollama
# Make permanent
setx PATH "%PATH%;C:\Users\<you>\AppData\Local\Programs\Ollama"
02

Set Up Python Environment in VS Code

Create a virtual environment and install dependencies in your project folder.

cmd
python -m venv venv
venv\Scripts\activate
pip install ollama rich
03

Pull Models

Ollama auto-detects your NVIDIA GPU. RTX 5070 with 8GB VRAM handles all three models.

cmd
ollama pull phi3:mini # 2.3 GB
ollama pull mistral:7b # 4.1 GB
ollama pull qwen2.5-coder:7b # 4.7 GB

# Verify all models are ready
ollama list

Models Tested
phi3:mini
Microsoft's Efficient Powerhouse
2.3 GB ¬∑ 3.8B params

Despite its small 3.8B parameter count, phi3:mini punches well above its weight. Strong focus on reasoning and instruction-following. Fits entirely in VRAM with room to spare ‚Äî enabling blazing fast inference.

mistral:7b
The Reliable All-Rounder
4.1 GB ¬∑ 7B params

Widely regarded as one of the best 7B models available. Uses grouped-query attention and sliding window attention. More thorough and accurate on complex reasoning tasks.

qwen2.5-coder:7b
The Code Specialist
4.7 GB ¬∑ 7B params

Purpose-built for software development ‚Äî trained on a massive corpus of code across dozens of programming languages. Best docstrings, edge case handling, and code explanation quality of the three.

Run It Yourself
python
import ollama, time
from rich.console import Console
from rich.panel import Panel

console = Console()
MODEL = "phi3:mini"

def run_test(label, prompt, category):
  console.rule(f"üß™ {category} ‚Äî {label}")
  start = time.time()
  response = ollama.chat(model=MODEL, messages=[{"role":"user","content":prompt}])
  elapsed = time.time() - start
  tokens = response.get('eval_count', 0)
  console.print(Panel(response['message']['content']))
  return {"time": round(elapsed,2), "tokens": tokens, "tok_per_sec": round(tokens/elapsed,2)}
