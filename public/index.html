Getting Started
01

Install Ollama

Download the Windows installer from ollama.com/download and run it. Installs silently, starts a background service on port 11434.

# Verify installation (new terminal)
ollama --version

# If not recognized, add to PATH (CMD)
set PATH=%PATH%;C:\Users\<you>\AppData\Local\Programs\Ollama
02

Set Up Python Environment in VS Code

Create a virtual environment and install dependencies in your project folder.

python -m venv venv
venv\Scripts\activate
pip install ollama rich
03

Pull Models

Ollama auto-detects your NVIDIA GPU ‚Äî no CUDA config needed. RTX 5070 with 8GB VRAM handles all three models.

ollama pull phi3:mini # 2.3 GB
ollama pull mistral:7b # 4.1 GB
ollama pull qwen2.5-coder:7b # 4.7 GB
